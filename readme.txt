
##########################################
# Weapon Specific Customization Files
##########################################
Each weapon can have specific aspects customized by the user. 

Weapon Specific customization files go here
C:\Program Files (x86)\Steam\steamapps\common\Team Fortress 2\tf\scripts

example file names:
tf_weapon_grenadelauncher.txt
tf_weapon_pipebomblauncher.txt
tf_weapon_rocketlauncher_directhit.txt
tf_weapon_sentry_revenge.txt
tf_weapon_shotgun_building_rescue.txt
tf_weapon_shotgun_soldier.txt
tf_weapon_sniperrifle.txt
tf_weapon_wrench.txt


##########################################
# Individual Class Configuration 
##########################################
Each individual class can have it's own .cfg that gets run when you switch classes. 

Put these files in the directory that applies to your config-

autoexec.cfg
sniper.cfg
soldier.cfg
spy.cfg
demoman.cfg
engineer.cfg
heavyweapons.cfg
medic.cfg
pyro.cfg
scout.cfg
modules.cfg (for mastercomfig)

If you are using mastercomfig - put those .cfg files into this directory:
C:\Program Files (x86)\Steam\steamapps\common\Team Fortress 2\tf\cfg\overrides

If you are NOT using mastercomfig - then put those .cfg files into this directory:
C:\Program Files (x86)\Steam\steamapps\common\Team Fortress 2\tf\cfg
